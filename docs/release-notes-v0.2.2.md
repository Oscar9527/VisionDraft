# VisionDraft v0.2.2

对比基线：`v0.2.1`

## 新增

- 分镜单标题支持实时预览编辑。
- 分镜单标题支持 X / Y 轴位置微调滑块。
- 批量模式下支持多镜头成组拖拽与移动。

## 改进

- 分镜单中的场标题改为合并整行并居中显示，场分隔更清晰。
- 项目库与导出页面去掉了一部分冗余说明文案，界面更干净。
- Windows 打包流程对缺失 `native_assets` 目录的情况更稳健，不再因为安装阶段缺目录而中断。

## 修复

- 修复分镜单标题输入后预览不即时刷新的问题。
- 修复清空标题后预览仍强制显示“分镜单”的问题。
- 修复 Windows 发布构建链路，确保新的桌面发行包可以正常生成。

## 已知限制

- 当前公开交付仍以 Windows 桌面版为主。
- Android 相关工作仍在开发中，本次发布不包含 Android 成品包。
