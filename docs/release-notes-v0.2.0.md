# VisionDraft v0.2.0

对比基线：`v0.1.0`

## 新增

- AI 分镜草案入口与本地服务商配置
- 分镜导出预览面板
- Excel 导出
- 批量行高调整
- 自定义列与列模板能力补强
- 导出品牌区基础配置

## 改进

- Windows 构建流程改为仓库内可复现脚本
- 分镜单切换为 A4 横向主版式
- 导出页支持更明确的输出类型选择
- 项目库支持手动选择项目创建路径与打开已有项目

## 修复

- 修复列菜单文字显示不全
- 修复分镜单预览横向适配错误
- 修复分镜单分页过早导致每页镜头数过低
- 修复编辑视口缩放与导出预览不同步的部分问题

## 已知限制

- 当前公开交付仍以 Windows 桌面为主
- 分镜单仍需继续提升无图模式下的版面密度
- “场”分组工作流将在下一阶段补齐
